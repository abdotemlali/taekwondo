# Tableau de Score Taekwondo

Une application web complète de tableau de score pour les compétitions sportives de Taekwondo, développée avec **React** et **Tailwind CSS 4**.

## Fonctionnalités

- 📺 **Affichage Public (Écran Géant)** : Interface ultra-premium (animations fluides, mode sombre, très haute visibilité)
- 🎯 **Console Arbitre (Tablette)** : Interface de contrôle claire et protégée par mot de passe (Code admin: `2026`)
- ⚙️ **Paramétrage souple** : Noms, nombre de rounds, durée, valeur des frappes configurables.
- 🔄 **Synchronisation en temps réel** : via un état géré localement dans le composant au travers d'un reducer.

## Démarrage rapide

1. Installez les dépendances :
   ```bash
   npm install
   ```
2. Lancez le serveur local :
   ```bash
   npm run dev
   ```

Créé par l'équipe de développement.
